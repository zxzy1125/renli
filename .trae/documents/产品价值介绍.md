## 一、产品定位

**人力代招公司**专用的 AI 招聘辅助工具，核心是"**AI 起草 + 人工审核**"半人工工作流。

- 📥 客户公司职位由管理员统一录入，AI 解析成结构化字段
- 📄 员工录入求职者简历，AI 自动识别画像/共同点/风险
- 🎯 选职位 + 选简历 → AI 生成匹配报告 + 18 条话术
- 📅 自动跟进回访，AI 生成作战卡片 + 深度分析
- 📊 管理员看团队漏斗、员工绩效、客户公司汇总

**核心理念**：AI 干脏活累活，员工干决策活。让每个员工都拥有 10 年资深顾问的脑子和效率。

## 二、给员工带来的优势

### 价值 1：职位秒懂，不再被表格绕晕

**场景**：客户发来几十行职位 Excel，传统方式要花 20 分钟读懂。

**工具价值**：
- 管理员录入后，AI 自动把杂乱原文解析成结构化字段
- 你看到的是：职位名、薪资、地点、职责、要求、加分项，清清楚楚
- 原始文本折叠保留，需要时展开核对
- **节省时间**：20 分钟 → 30 秒

### 价值 2：简历一看就懂，破冰有抓手

**场景**：求职者简历到手，不知从哪聊起。

**工具价值**：
- AI 解析简历时自动挖掘 **4 类共同点**：
  - 🎓 校友（同校）
  - 🏠 老乡（同地）
  - 🏢 前公司（同公司）
  - 🎯 爱好（同好）
- 直接生成破冰话术："张哥，看到你也是 xx 毕业的，咱们校友！"
- 同时自动识别 **8 类风险求职者**（套话、骗资料、恶意举报），红色警告
- **效果**：破冰成功率提升 50%+

### 价值 3：18 条话术，覆盖沟通全链路

**场景**：以前跟求职者沟通全靠临场发挥，质量参差不齐。

**工具价值**：
- 选定「1 个职位 + 1 份简历」→ AI 自动生成 **18 条话术**：

  | | 微信文字 | 电话话术 | 站内信 |
  |---|---|---|---|
  | 触达开场白 | ✅ | ✅ | ✅ |
  | 职位介绍 | ✅ | ✅ | ✅ |
  | 疑虑应对 | ✅ | ✅ | ✅ |
  | 面试邀约 | ✅ | ✅ | ✅ |
  | 薪资沟通 | ✅ | ✅ | ✅ |
  | Offer 促签 | ✅ | ✅ | ✅ |

- **每条话术都是为这个求职者量身定制的**，不是模板填空
- 所有话术**注入了 BOSS 规范硬约束**：不暴露甲方、不违反劳动法、符合平台规则
- **半人工审核**：AI 出草稿，你审核微调后一键复制发送
- **效果**：回复率提升 30-50%

### 价值 4：跟进不漏，一个求职者推多个职位

**场景**：传统方式，求职者聊完没下文就忘了，线索浪费。

**工具价值**：
- 为每个求职者创建**回访计划**（一次性/周期性/自定义）
- 系统自动提醒：**今日待回访 / 逾期未回访**
- 第一次推职位 A 不成 → 5 天后回访推职位 B → 再 5 天推职位 C
- **一个求职者最多能推 5 个职位**，大幅提升单线索入职率
- **效果**：单线索转化率提升 2-3 倍

### 价值 5：AI 陪你打仗，新手也能做出老手业绩

**场景**：新人不知道怎么应对求职者的各种顾虑，容易谈崩。

**工具价值**：

**回访前**：AI 生成「作战卡片」
- 求职者画像速览
- 本次回访目标
- 预判的求职者顾虑（3-5 条）
- 3 套开场话术可选
- 推荐提问引导求职者说出真实想法
- 转化可能性预判

**回访后**：AI 生成「深度分析报告」
- 顾虑解析（深度分析每个顾虑的根源）
- 跟进策略建议（4-6 条，按优先级）
- 转化可能性更新（升/降 + 原因）
- 下次回访时间 + 重点 + 准备物料
- **针对每个顾虑一键生成 3 渠道应对话术**

**效果**：新人 1 个月达到老手 6 个月水平

### 价值 6：合规保障，账号不封禁

**场景**：BOSS 直聘规则复杂，稍不注意就被封号。

**工具价值**：
- 所有话术自动注入 **BOSS 硬约束 7 条**：
  - 严禁暴露甲方公司名
  - 严禁违反劳动法表述
  - 严禁辱骂阴阳求职者
  - 最后一条信息必须 HR 收尾
  - 难缠求职者用"不好意思打扰了"退出
- 内置完整 **BOSS 规范操作指南**（随时查阅）
- 内置 **HR 话术方法论知识库**（9 类画像破冰策略）
- **效果**：账号封禁风险降低 90%

### 价值 7：BOSS 发布文案一键生成

**场景**：每次发职位都要绞尽脑汁写吸引人的描述。

**工具价值**：
- 职位详情页点「生成 BOSS 发布文案」
- AI 生成 3 套不同风格（诱惑型/神秘型/专业型）
- 模糊化处理引发好奇，但不影响求职者咨询
- 一键复制，直接去 BOSS 发布
- **效果**：职位咨询量提升 2-3 倍

## 三、给公司管理带来的优势

### 价值 1：撞单自动检测，杜绝内耗

**场景**：员工 A 和员工 B 同时跟进同一个求职者，互相抢线索，内耗严重。

**工具价值**：
- 员工录入简历时，系统自动用手机号哈希 + 邮箱 + 姓名+公司做全团队撞单检测
- 检测到撞单 → 提示员工 + 自动通知管理员
- 管理员在撞单管理页处理：
  - 归属 A（B 的简历软删除）
  - 归属 B（A 的简历软删除）
  - 共享独立跟进（各维护各的）
  - 标记误报（同名不同人）
- **效果**：撞单内耗降低 95%

### 价值 2：手机号脱敏，数据安全合规

**场景**：员工离职带走求职者数据，造成客户流失。

**工具价值**：
- 手机号脱敏存储（138\*\*\*\*5678），员工看不到完整号码
- 系统内部用 SHA256 哈希做撞单检测
- 简历数据加密存储 + 访问日志
- 删除操作走软删除 + 回收站（30 天可恢复）
- **效果**：数据泄露风险降低 80%

### 价值 3：权限分级，数据隔离

**场景**：员工能看到所有人的数据，造成数据混乱和泄露。

**工具价值**：

| 数据类型 | 录入权 | 可见范围 |
|---------|--------|---------|
| 职位库 | 仅管理员 | 团队共享 |
| 客户公司 | 仅管理员 | 团队共享 |
| 求职者简历 | 全员 | 仅本人 + 管理员 |
| 匹配/话术/跟进 | 全员 | 仅本人 + 管理员 |
| 撞单管理 | 自动 | 仅管理员 |
| 团队管理 | 仅管理员 | 仅管理员 |
| AI 配置 | 仅管理员 | 仅管理员（API Key 脱敏） |

- **效果**：数据安全 + 隐私保护 + 权责清晰

### 价值 4：操作审计，责任可追溯

**场景**：出问题找不到是谁干的。

**工具价值**：
- 所有数据变更记录操作人、时间、变更内容
- 主管可查询操作日志
- 敏感操作（删除、导出）单独审计
- 简历授权使用（求职者同意条款）
- 简历保留期限（如 2 年自动清理）
- **效果**：责任 100% 可追溯

### 价值 5：客户公司集中管理

**场景**：客户公司信息散落在各员工手里，离职带走客户。

**工具价值**：
- 客户公司由管理员统一录入，团队共享
- 一个客户 → 多个职位关联
- 客户公司维度的转化漏斗
- 客户公司维度的结算报表（按入职人数计费）
- 客户健康度（活跃度、转化率、续约风险）
- **效果**：客户资产公司化，不依赖个人

### 价值 6：团队报表，一眼看穿

**场景**：管理员不知道团队每个人在干什么，业绩如何。

**工具价值**：
- **转化漏斗**：团队整体 / 个人（咨询→邀面→面过→Offer→入职）
- **顾问绩效**：按月/季度（入职人数、转化率、平均周期）
- **客户公司报表**：按客户（A 客户：3 职位 → 入职 8 人）
- **职位热度**：哪些职位咨询多但转化低
- **话术效果**：哪些话术对应的转化率高
- 可视化：漏斗图、折线图、柱状图、热力图
- 可导出 Excel + 定时邮件推送周报
- **效果**：管理从"凭感觉"变成"看数据"

### 价值 7：风险预警，主动管理

**场景**：等问题爆发了才发现，为时已晚。

**工具价值**：
- A 客户职位 30 天无入职 → 红色预警
- 张三本周转化率环比下降 20% → 黄色预警
- 5 个面试今天进行 → 提醒跟进
- 3 个简历未解析 → 待处理任务
- 2 个匹配待审核话术 → 待处理任务
- **效果**：从被动救火变成主动预防

### 价值 8：标准化运营，不依赖个人能力

**场景**：公司业绩依赖几个销冠，销冠离职业绩腰斩。

**工具价值**：
- AI 话术 + 方法论沉淀为公司资产
- 新人按系统流程走，1 个月达到老手水平
- 优质话术存入团队话术库，全员复用
- 提示词模板可自定义，沉淀公司经验
- **效果**：业绩稳定性提升 3 倍，不再依赖个人

### 价值 9：BOSS 平台合规运营

**场景**：员工违规操作导致公司账号被封，全公司停摆。

**工具价值**：
- 所有话术自动注入 BOSS 合规约束
- 风险求职者自动识别，避免被举报
- 操作日志全程留痕，可追溯责任
- 内置完整 BOSS 规范知识库，新人必修
- **效果**：账号被封风险降低 90%，公司业务连续性保障

## 四、员工绩效管理优势

### 1. 多维度绩效指标自动统计

**传统方式**：管理员月底手动统计每个人入职多少、转化率多少，耗时且不准确。

**工具价值**：系统自动统计每个员工的：

| 指标 | 说明 | 用途 |
|------|------|------|
| 简历录入数 | 本月新增简历数 | 衡量线索开拓能力 |
| 匹配创建数 | 本月发起匹配数 | 衡量匹配主动性 |
| 邀面数 | 推进到「已邀面试」数 | 衡量初筛能力 |
| 面过数 | 推进到「面试通过」数 | 衡量岗位匹配精度 |
| Offer 数 | 推进到「已发 Offer」数 | 衡量谈薪促签能力 |
| 入职数 | 推进到「已入职」数 | **核心业绩指标** |
| 转化率 | 入职数 / 匹配数 | 衡量整体效率 |
| 平均周期 | 从咨询到入职平均天数 | 衡量推进效率 |
| 回访次数 | 本月完成回访次数 | 衡量跟进勤奋度 |
| 话术使用量 | 本月使用话术条数 | 衡量工具使用度 |

**效果**：绩效统计从"月底加班 2 天" → "实时一键查看"

### 2. 员工排行榜，激发竞争

**工具价值**：
- 本月入职数排行榜（实时更新）
- 转化率排行榜
- 回访勤奋度排行榜
- 可按部门/全公司筛选
- Top 3 自动高亮，激发良性竞争
- **效果**：团队战斗力提升 30%

### 3. 个人能力诊断，精准辅导

**场景**：管理员不知道员工哪里弱，怎么培训。

**工具价值**：系统自动诊断每个员工的薄弱环节：

| 诊断维度 | 数据来源 | 辅导建议 |
|---------|---------|---------|
| 简历录入少 | 简历数 | 加强线索开拓 |
| 匹配多但邀面少 | 匹配数 vs 邀面数 | 提升话术质量 |
| 邀面多但面过少 | 邀面数 vs 面过数 | 优化岗位匹配精度 |
| 面过多但 Offer 少 | 面过数 vs Offer 数 | 加强谈薪技巧 |
| Offer 多但入职少 | Offer 数 vs 入职数 | 强化 Offer 促签 |
| 回访次数少 | 回访记录 | 加强持续跟进 |
| 转化周期长 | 平均周期 | 提升推进效率 |

- 管理员根据诊断结果定向辅导
- **效果**：员工成长速度提升 2 倍

### 4. 业绩趋势分析，预警异常

**工具价值**：
- 每个员工的业绩趋势图（折线图）
- 环比/同比变化自动计算
- 异常下降自动预警：
  - 张三本周转化率环比下降 20% → 黄色预警
  - 李四本月 0 入职（上月 3 入职）→ 红色预警
- 管理员及时介入辅导
- **效果**：业绩波动提前 2 周发现

### 5. 客户公司维度绩效

**场景**：客户 A 投诉没人推他们的职位，但不知道是谁的责任。

**工具价值**：
- 按客户公司维度看绩效：
  - 客户 A → 员工张三跟进 → 3 入职
  - 客户 A → 员工李四跟进 → 0 入职
- 一眼看出谁在哪个客户上表现好/差
- 客户健康度（活跃度、转化率、续约风险）
- **效果**：客户满意度提升 40%

### 6. 绩效报表导出与推送

**工具价值**：
- 一键导出 Excel 绩效报表
- 定时邮件推送周报/月报到管理员邮箱
- 含：员工排名、转化漏斗、异常预警、辅导建议
- 可自定义报表模板
- **效果**：管理效率提升 5 倍

### 7. 数据驱动的人事决策

**工具价值**：基于数据做人事决策：
- 晋升：转化率持续 Top 3 的员工
- 培训：转化率持续垫底的员工
- 调岗：某客户维度长期 0 入职的员工
- 淘汰：3 个月无入职且诊断无改善的员工
- 激励：本月 Top 1 现金奖励
- **效果**：人事决策从"凭感觉"变成"看数据"

### 8. 团队协作透明化

**工具价值**：
- 团队整体漏斗：一眼看穿团队瓶颈在哪一环
- 个人漏斗对比：找出团队短板
- 协作记录：员工间的协作（@、评论）沉淀
- 优质话术共享：好话术全员复用
- **效果**：团队协作效率提升 40%
